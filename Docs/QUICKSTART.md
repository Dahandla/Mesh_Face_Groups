# Mesh Face Groups Free Quickstart

https://github.com/Dahandla/Mesh_Face_Groups/blob/0e26df823d34d1e4a4a239181360bb84c66f4d8c/Mesh_Face_Groups.png

1. Install `dist/mesh_face_groups_free_v1.3.0.zip` in Blender.
2. Enable `Mesh Face Groups Free`.
3. Select a mesh object.
4. Switch to Object Mode.
5. Open `3D Viewport > Sidebar (N) > Face Groups`.
6. Enable `Live Preview`.
7. Tune `Angle Threshold`.
8. Click `Refresh Face Groups`.

The install zip is Blender-ready and contains `blender_manifest.toml` and `__init__.py` at zip root.

## Cleanup

- Enable `Remove Small Islands` to reduce tiny colored speckles.
- Enable `Remove Floating Islands` to ignore small disconnected debris during grouping.
- Click `Delete Floating Islands` to permanently remove matching debris from the mesh.
