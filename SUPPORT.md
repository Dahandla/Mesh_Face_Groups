# Support

Before reporting an issue, check:

- Blender version is 4.5 or newer.
- A mesh object is selected.
- The object is in Object Mode.
- The issue reproduces after restarting Blender.

Include:

```text
Blender version:
Operating system:
Free add-on version:
Mesh source/type:
Approximate face count:
Action being run:
Error message:
```

For separated parts, watertight output, shell thickness, exploded views, or batch workflows, use Mesh Face Groups Pro.
