# Installation

## Install From Zip

1. Open Blender.
2. Go to `Edit > Preferences > Add-ons`.
3. Click `Install...`.
4. Select `dist/mesh_face_groups_free_v1.3.0.zip`.
5. Enable `Mesh Face Groups Free`.
6. Open `3D Viewport > Sidebar (N) > Face Groups > Face Group Tools Free`.

The install zip is packaged for Blender's direct install flow and includes:
- `blender_manifest.toml` at the zip root.
- `__init__.py` at the zip root.

## Install From Folder

Copy the `mesh_face_groups_free` folder into your Blender add-ons directory, restart Blender, and enable `Mesh Face Groups Free`.

## Uninstall

Disable `Mesh Face Groups Free` in Blender's add-on preferences, then remove the installed `mesh_face_groups_free` folder if needed.
